# Games Store

O presente projeto de banco de dados irá modelar a área de e-commerce de jogos eletrônicos. O escopo deste projeto aborda boa parte das entidades de um négocio real de comércio de jogos online. Algumas delas são: os usuários, as compras que os mesmos podem fazer, os jogos que são comprados, gêneros e classificações indicativas (que são próprias dos jogos), editoras, desenvolvedoras e distribuidoras de cada um dos jogos.
